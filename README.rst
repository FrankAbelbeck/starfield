Description
===========

This is a simple starfield demo for the card10 badge.
Written in MicroPython, prepared to be integrated as badge app.

Controls:

Top right button
	Toggle text output (warp speed, number of stars, load).

Bottom left button
	Decrease warp speed.

Bottom right button
	Increase warp speed.
